Для установки отдельных счетчиков для поддоменов необходимо создать .js файлы с названием поддомена содержащие код счетчика.

Напрмер файл тула.js c cодержанием // TULA_COUNTER_CODE
И так по каждому домену.
ЛОгика CounterManager настроена следующим образом: Если файла для поддомена не существует но сайт открыт на поддомене - будет установлен счетчик по умолчанию, то-есть счётчик из файла default.js
